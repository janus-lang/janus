# std.quant: The Quantum Arsenal

**Philosophy:** Quantum computing is not a gimmick; it is the baseline for optimization and simulation by 2030. This module provides primitives to avoid 'quantum winter' regrets.

**Key Modules:**
*   `qbit`: Qubit simulation (statevector/matrix).
*   `circuit`: Gate-based circuits (Hadamard, CNOT).
*   `algo`: Grover/Shor/VQE.

**Profile Gate:** `:quantum`
