# std.neuro: Neuromorphic & Brain-Inspired

**Philosophy:** Silicon brains (SNNs) enable 100x energy efficiency in edge AI. This module supports decentralized learning without data centers.

**Key Modules:**
*   `snn`: Spiking neurons (LIF model).
*   `plasticity`: Hebbian learning rules.
*   `swarm`: Federated learning over P2P.
