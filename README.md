Janus is not a toy. It is a weapon against software decay.

We built Janus because we were tired of choosing between the safety of a straitjacket and the freedom of a foot-gun. We wanted:
*   **Scientific Rigor** without the bureaucratic overhead.
*   **System Control** without the memory leaks.
*   **Artificial Intelligence** that isn't just a wrapper around a Python API.

Janus is a **Sovereign Systems Language**. It compiles to native machine code via LLVM, runs JIT for rapid prototyping, and enforces deterministic resource management.

## ‚ö° v0.2.0 Capabilities

This release marks the point of no return. We are Turing Complete, we are loud, and we are precise.

*   **Sovereign Numerics**: `f64` precision and `VectorF64` dynamic handles. Math that doesn't apologize.
*   **Deterministic Safety**: `defer` cleanups. We don't garbage collect; we take out the trash ourselves.
*   **Dual-State Existence**:
    *   `janus run`: Immediate JIT execution (Interpreter).
    *   `janus build`: Optimized LLVM Native Compilation (AOT).
    *   **Parity**: The code behaves the same in the Bazaar as it does in the Cathedral.
*   **String Handles**: Proper heap-managed strings. No more `char*` soup.
