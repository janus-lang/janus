## âš ï¸ Known Issues (Technical Debt)

### Catalogued for v0.2.0

1. **String Byte Alignment**
   - Minor debug warnings regarding string length calculations (23 vs 22 bytes) in `lowerStringLiteral`
   - **Impact**: None - does not affect runtime behavior
   - **Root Cause**: Quote removal edge case in string literal processing

2. **Runtime Dependency**
   - Current `janus_rt` relies on thin C shim (`janus_rt.c`) linking to `libc`
   - **Planned Fix**: Phase 2 will replace with direct Zig/Assembly syscalls
   - **Goal**: Zero external dependencies (true sovereignty)

3. **Profile Limitations**
   - `while` loop implemented but blocked in `:core` profile
   - **Rationale**: Enforces functional purity (use recursion instead)
   - **Status**: Feature, not bug (`:core` is the "Monastery")

4. **Integration Test Updates**
   - Some older tests need updates for multi-function API
   - **Core Functionality**: Unaffected (E2E test passes)
